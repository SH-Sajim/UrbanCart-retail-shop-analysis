# UrbanCart Retail Shop Data Analysis

This project analyzes retail transaction data from UrbanCart to understand
sales performance, customer behavior, and cross-selling opportunities using SQL.

## Tools Used
- PostgreSQL
- SQL
- Excel/Google Sheet

## Key Business Insights
• Identified top-performing products, categories, and cities driving overall revenue
• Analyzed repeat purchase behavior and customer lifecycle trends
• Discovered high-value product combinations suitable for bundling and cross-selling
• Highlighted potential inventory stock-out risks based on demand versus stock levels
• Observed seasonal patterns in order volume and customer demand
• Evaluated payment method preferences and their impact on order completion and order value

## Project Files
- `report/UrbanCart_Retail_Analysis.pdf` – Final analysis report
- `sql/urbancart_queries.sql` – SQL queries used for analysis
